# lita-env

A plugin for Lita to store details of your environments. 

## Installation

Add lita-env to your Lita instance's Gemfile:

``` ruby
gem "lita-env"
```


## Configuration


## Usage

Add a new environment `lita env add webserver`
List all environments `lita env list`
